## 1. 앱 컨셉

### 앱 이름
**수상한 AI 연구실** (Suspicious AI Lab)

### 슬로건
> "이 연구실에서는 무슨 실험이 벌어지고 있는 걸까?"

### 서브 카피
- "AI가 당신을 분석합니다 — 수배전단부터 평행우주까지"
- "수상한 실험에 참여하시겠습니까?"

### 톤앤매너
| 항목 | 방향 |
|------|------|
| **말투** | 연구원/수사관 톤. "피험자님, 흥미로운 결과가 나왔습니다", "수상한 데이터를 발견했습니다" |
| **분위기** | 레이튼 교수 퍼즐/미스터리 테마. 수상하지만 유쾌한 느낌 |
| **시각** | 다크 사이버 + 네온 그린 (검토 중). 해커 연구실 / 사이버펑크 미학 |
| **핵심 가치** | 재미 > 정확성. "수상한 AI 연구실의 실험 결과입니다" 면책 문구 |
| **타겟** | 10대 후반 ~ 20대 대학생/직장인. SNS 공유 중심 |

### 브랜딩 컬러 팔레트 (검토 중 - 4개 후보)

**후보 1: 다크 사이버 + 네온 그린** (선정)
```
Background:  #0a0e17 (딥 다크 네이비)
Card BG:     #1a1f2e (다크 블루그레이)
Primary:     #00ff88 (네온 그린)
Secondary:   #22d3ee (시안)
Text:        #e2e8f0 (라이트 그레이)
Button:      #059669 → #047857 (그린 그라디언트)
```

**후보 2: 딥 틸 + 일렉트릭 퍼플**
**후보 3: 다크 네이비 + 코랄/핑크**
**후보 4: 미드나잇 블랙 + 크롬/실버**

(프리뷰: `streamlit run color_preview.py`)

## 2. 타겟 유저

### 주요 페르소나

#### 페르소나 A: "심심한 대학생 지은" (22세)
- **상황**: 시험 기간인데 공부하기 싫어서 SNS 하는 중
- **니즈**: 가볍게 즐길 수 있는 콘텐츠, 친구에게 공유할만한 재밌는 결과
- **시나리오**: 인스타 스토리에서 친구가 올린 수배전단 결과 보고 → 들어와서 자기도 해봄 → 프로파일링도 해봄 → 결과 캡처해서 스토리에 올림
- **핵심 동기**: 재미, SNS 콘텐츠 소재

#### 페르소나 B: "호기심 많은 직장인 현우" (28세)
- **상황**: 점심시간에 동료들과 잡담 중
- **니즈**: 대화 소재, 가벼운 오락
- **시나리오**: "이런 앱 있대" 하고 동료한테 보여줌 → 다같이 수배전단 만들어봄 → 결과 비교하며 깔깔 웃음
- **핵심 동기**: 대화 소재, 그룹 활동

### 사용 피크 타임
- 점심시간 (12:00-13:00)
- 퇴근 후 (18:00-22:00)
- 주말 오후

## 3. 전체 콘텐츠 구성

### 메인 실험 (수상한 실험실) - 4개

| # | 콘텐츠 | 파일 | 입력 | 출력 | 기술 |
|---|--------|------|------|------|------|
| 1 | 🔍 **수배전단 생성기** | `wanted_poster.py` | 사진/카메라/텍스트 묘사 | 수배포스터 일러스트 + 죄목/위험등급/현상금/특이사항 | Vision + GPT + DALL-E |
| 2 | 🌀 **평행우주 연구소** | `parallel_universe.py` | 이름 + 생년월일 + 분기점 퀴즈 3개 | 평행우주 프로필(직업/성격/일상/연봉) + 초상화 + 괴리율 | GPT + DALL-E + Plotly |
| 3 | 🧠 **심리 프로파일링** | `profiling.py` | 극한선택 퀴즈 6개 | FBI식 보고서(유형명/위험등급/능력치/약점/파트너유형) + 캐릭터 | GPT + DALL-E + Plotly |
| 4 | ❓ **추리 퀴즈** | `mystery_quiz.py` | 난이도 선택 → 단서 확인 → 범인 선택 | 정답판정 + 해설 + 추리력등급 + 사건현장 일러스트 | GPT + DALL-E |

## 5. 기능별 상세 기획

### 5-1. 수배전단 생성기

**UX 플로우:**
```
[Step 1: 입력 방식 선택]
  ├─ 📁 사진 업로드
  ├─ 📷 카메라 촬영
  └─ 📝 텍스트로 묘사

[Step 2: 분석 & 생성]
  ├─ Vision API로 외모 특징 추출 (사진인 경우)
  ├─ GPT로 수배 정보 생성 (죄목/위험등급/현상금/특이사항)
  └─ DALL-E로 수배전단 일러스트

[Step 3: 결과 표시]
  ├─ 수배전단 카드 (일러스트 + 정보)
  ├─ 외모 특징 분석
  ├─ 공유 & 다운로드
  └─ 다시 만들기
```

### 5-2. 평행우주 연구소

**UX 플로우:**
```
[Step 1: 기본 정보]
  ├─ 이름
  └─ 생년월일

[Step 2: 분기점 퀴즈 3개]
  ├─ "대학 전공을 다시 고른다면?"
  ├─ "10년 전으로 돌아간다면?"
  └─ "다른 나라에서 태어났다면?"

[Step 3: 결과]
  ├─ 평행우주 프로필 (직업/성격/일상/연봉)
  ├─ 괴리율 (%)
  ├─ 레이더 차트 (Plotly)
  ├─ DALL-E 초상화
  └─ 공유 & 다운로드
```

### 5-3. 심리 프로파일링

**UX 플로우:**
```
[Step 1: 극한선택 퀴즈 6개]
  └─ "무인도에 하나만?", "범인을 목격했다면?" 등

[Step 2: FBI식 프로파일 보고서]
  ├─ 유형명 + 위험등급
  ├─ 능력치 레이더 차트 (Plotly)
  ├─ 약점 & 파트너 유형
  ├─ DALL-E 캐릭터 일러스트
  └─ 공유 & 다운로드
```

### 5-4. 추리 퀴즈

**UX 플로우:**
```
[Step 1: 난이도 선택]
  ├─ 초급 (단서 3개)
  ├─ 중급 (단서 2개)
  └─ 고급 (단서 1개)

[Step 2: 사건 제시]
  ├─ GPT가 미니 미스터리 시나리오 생성
  ├─ 4명 용의자 프로필
  └─ 단서 순차 공개

[Step 3: 범인 선택 → 판정]
  ├─ 정답/오답 + 해설
  ├─ 사용한 단서 수에 따른 점수
  ├─ 추리력 등급
  ├─ DALL-E 사건현장 일러스트
  └─ 공유 & 다운로드
```

### 5-5~8. 레거시 콘텐츠 (타로/관상/전생/뉴스웹툰)

기존 기능 유지. 변경사항:
- `show_other_features()` → `show_other_features_legacy()` 사용
- `show_stamp_sidebar()` 제거
- 사이드바 "봉인된 실험 기록" 섹션으로 이동
